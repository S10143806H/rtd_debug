[RTL8722CSM] [RTL8722DM] WiFi - WiFi 掃描¶

材料準備

  • Ameba x 1

範例說明

WiFi掃描功能可以幫助我們快速發現周圍有哪些WiFi網絡。

該範例不需要任何其他硬件，因此只需複製以下代碼並將其粘貼到REPL中即可查看其作用。

1from wireless import WLAN
2wifi = WLAN(mode = WLAN.STA)
3wifi.scan()
